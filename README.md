# 🎙️ Voice Assistant using Python

Python me banaya hua ek simple voice assistant jo mic se sunke commands execute karta hai.

## 🚀 Demo
https://github.com/Manju-0094/Voice_assistant-python/blob/main/demo.mp4
> Note: Abhi video add karni hai. Screen record karke `demo.mp4` naam se upload kar dena.

## ✨ Features
- **Voice Commands**: "youtube kholo", "google kholo", "time kya hai"
- **Text-to-Speech**: Assistant jawab bolke deta hai
- **Speech Recognition**: Google API use karke voice ko text me convert karta hai

## 🛠️ Tech Stack
- Python 3.13
- sounddevice - Mic se audio record karne ke liye
- SpeechRecognition - Voice ko text me badalne ke liye  
- pyttsx3 - Text ko voice me badalne ke liye
